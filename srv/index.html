Depoyu eklemek için:
curl https://sulincix.github.io/debian-repo-template/srv/stable/dists/stable/Release.key | apt-key add -
echo "deb https://sulincix.github.io/debian-repo-template/srv/stable/ stable main contrib non-free" > /etc/apt/sources.list.d/test.list
Ardından indexi güncellemek için:
apt-get update --allow-insecure-repositories
Son olarak bu depodan foo paketini kurmak için:
apt-get install foo --allow-unauthenticated
