Depoyu eklemek için:
echo "deb https://sulincix.github.io/debian-repo-template/srv/pardus/ main contrib non-free" > /etc/apt/sources.list.d/pardus.list
Ardından indexi güncellemek için:
apt-get update --allow-insecure-repositories
Son olarak bu depodan foo paketini kurmak için:
apt-get install foo --allow-unauthenticated
